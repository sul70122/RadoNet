FTTH系列是RadoNet的特色产品，用的是日本真家宽线路。

- 2TB起步流量，部分套餐给到10TB甚至40TB

**实际体验**：从多个测评来看，这个系列的流媒体解锁能力确实强。Netflix日本区、Hulu、AbemaTV、TVer这些日本本土平台都能正常观看。而且因为是家宽IP，被封的概率很低。

缺点也很明显：**回国延迟较高**，基本在150-200ms以上，而且线路会绕。如果你主要是国内用户访问，体验可能不太理想。但如果主要用途是看日本流媒体、做落地机，那这个延迟完全可以接受。

价格在10-15美元/月之间，考虑到是真家宽IP，这个价格算是合理的。最大的问题是**经常缺货**，想买得盯着官网，有货就赶紧下单。
